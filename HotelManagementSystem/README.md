# Hotel Management System

A console-based Hotel Management System developed using C#.

## Description

This project simulates a simple hotel management system that allows hotel staff to register guests, manage room bookings, calculate bills, apply discounts, generate receipts, and track loyalty points through a menu-driven console application.

## Features

- Register new guests
- View guest information
- Check-in guests
- Check-out and generate bills
- Apply discounts
- Upgrade room types
- Add room service notes
- Search guests by name
- Calculate loyalty points
- Print receipts
- Edit guest information

## Technologies Used

- C#
- .NET Console Application

## Programming Concepts Demonstrated

- Variables and data types
- Conditional statements
- Switch statements
- Loops
- String manipulation
- DateTime operations
- Random number generation
- Mathematical functions
- Input validation
- Receipt generation
