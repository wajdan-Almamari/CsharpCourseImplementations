# Arrays Practice Task Sheet

A C# console application that demonstrates different array operations through practical programming exercises.

## Description

This project contains ten array-based programming problems designed to practice core C# concepts such as array manipulation, sorting, searching, looping, calculations, and data analysis.

## Implemented Problems

### Problem 1: Temperature Log

* Store and display weekly temperature readings
* Use array length to count readings

### Problem 2: Student Score Board

* Store student scores
* Reverse array order
* Display results using foreach

### Problem 3: Product Price Finder

* Store product prices
* Search for a specific price using Array.IndexOf()

### Problem 4: Race Finish Times

* Store race finish times
* Sort finish times
* Display participant count

### Problem 5: Classroom Grade Report

* Sort grades
* Display rankings
* Reverse grades into descending order

### Problem 6: Warehouse Inventory Check

* Calculate total stock
* Calculate average stock
* Search for inventory quantities

### Problem 7: Library Book Shelf Scanner

* Sort copy counts
* Find highest inventory count
* Detect unavailable books

### Problem 8: Sales Performance Analyzer

* Analyze monthly revenue
* Calculate best and worst revenue
* Calculate average monthly revenue

### Problem 9: Flight Seat Allocation Display

* Store flight seat numbers
* Sort seat numbers
* Search for a target seat
* Display sorted and reversed seat allocations

### Problem 10: Hospital Patient Priority Queue

* Store patient severity scores
* Generate triage priority rankings
* Calculate median severity score
* Count critical cases
* Search for severity levels
