# Arrays Practice Task Sheet

A C# console application that demonstrates different array operations through practical programming exercises.

## Description

This project contains multiple array-based programming problems designed to practice core C# concepts such as array manipulation, sorting, searching, looping, and data analysis.

## Implemented Problems

### Problem 1: Temperature Log
- Store and display weekly temperature readings
- Use array length to count readings

### Problem 2: Student Score Board
- Display student scores
- Reverse array order

### Problem 3: Product Price Finder
- Search for a specific price using Array.IndexOf()

### Problem 4: Race Finish Times
- Sort runner finish times
- Display participant count

### Problem 5: Classroom Grade Report
- Sort grades
- Display rankings
- Reverse array order

### Problem 6: Warehouse Inventory Check
- Calculate total stock
- Calculate average stock
- Search for inventory quantities

### Problem 7: Library Book Shelf Scanner
- Sort copy counts
- Find highest inventory
- Detect unavailable books

### Problem 8: Sales Performance Analyzer
- Analyze monthly revenue
- Calculate best and worst revenue
- Calculate average monthly revenue
