# Library Management System

A console-based Library Management System developed using C#.

## Description

This project simulates a simple library management system that allows users to register members, manage books, borrow and return books, calculate fines, apply discounts, and generate reports through a menu-driven console application.

## Features

- Register library members
- Display member profiles
- Register books
- Search books by title
- Borrow books
- Return books
- Calculate late fines
- Apply member discounts
- Check borrowing eligibility
- Generate member IDs
- Display book details
- Calculate renewal fees
- Update member email
- Generate session summaries

## Technologies Used

- C#
- .NET Console Application
