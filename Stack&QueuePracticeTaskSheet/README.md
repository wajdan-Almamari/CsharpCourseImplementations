# Stack & Queue Practice

This repository contains a collection of C# exercises and practical implementations demonstrating the use of **Stack** and **Queue** data structures.

The project was completed as part of my C# learning journey to strengthen my understanding of collections, problem-solving, and algorithm design.

---

## Features

- Browser History Tracker
- Hotel Check-In Queue
- Text Editor Undo System
- Hospital Emergency Room Triage
- Parenthesis Validator
- Print Spooler
- Reverse a Sentence
- Multi-Level Undo with Redo
- Ticket Counter Simulation
- Order Processing Pipeline

---

## Technologies

- C#
- .NET
- Stack
- Queue
- Collections
- Console Application

---

## Learning Objectives

- Understand Stack (LIFO) and Queue (FIFO)
- Practice real-world implementations
- Improve problem-solving skills
- Apply collection operations in C#
