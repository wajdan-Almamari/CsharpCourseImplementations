# Clinic Management System (CMS)

A console-based Clinic Management System developed using C#.

## Description

This project allows users to manage patient records through a simple menu-driven console application. Users can add, view, edit, delete, and generate reports for patient information.

## Features

* Add new patient information
* View a single patient
* View all patients
* Edit patient information
* Delete patient records
* Generate patient reports
* Menu-driven navigation

## Technologies Used

* C#
* .NET Console Application

## Project Purpose

This project was created to practice C# programming concepts, including:

* Variables and data types
* Conditional statements
* Switch statements
* Loops
* User input and validation
* Console application development

## Future Improvements

* Support multiple patients
* Add doctor management
* Add appointment scheduling
* Store data in files or databases
* Improve input validation
